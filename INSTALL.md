# Cómo instalar LifeFlow ahora mismo

## iPhone / iPad (sin App Store)
1. Abre **Safari** (debe ser Safari, no Chrome)
2. Ve a: **growthplayers.vercel.app**
3. Toca el ícono de compartir (cuadrado con flecha ↑)
4. Desliza hacia abajo → toca **"Agregar a pantalla de inicio"**
5. Toca **"Agregar"** → LifeFlow aparece en tu homescreen
6. Abre desde el ícono → experiencia fullscreen completa ✅

---

## Android

### Opción A — Instala el APK directo (recomendado):
1. Descarga desde: **[LINK DEL APK ← agregar cuando esté listo]**
2. Abre el archivo descargado
3. Si pide permiso → Configuración → Permitir esta fuente → Instalar
4. LifeFlow aparece en tu pantalla de inicio ✅

### Opción B — Desde Chrome (PWA):
1. Abre **Chrome**
2. Ve a: **growthplayers.vercel.app**
3. Toca los tres puntos (⋮) → **"Instalar app"** o **"Agregar a inicio"**
4. Confirma → ícono en homescreen ✅

---

## Computadora (Mac / Windows / Linux)
1. Abre **Chrome** o **Edge**
2. Ve a: **growthplayers.vercel.app**
3. En la barra de direcciones → ícono de instalar (⊕)
4. Toca **"Instalar LifeFlow"** → funciona como app de escritorio ✅
